Características clave

Tecnología moderna pensada para el negocio argentino

Cloud nativo

Accedé desde cualquier dispositivo sin instalar nada. Docker + PostgreSQL.

Multi-tenant

Cada cliente con su base de datos propia. Aislamiento total de datos.

Ticket térmico

Impresión directa de tickets. Compatible con impresoras térmicas estándar.

WhatsApp + Email

Envío de comprobantes y recordatorios por WhatsApp y correo electrónico.

Mercado Pago

Link de pago y QR. El webhook marca automáticamente como pagado.

Backup automático

Backups programados a Google Drive con service account.

Dark mode

Interfaz moderna con modo oscuro y claro. Personalizable por usuario.

Exportación Excel y PDF

Exportá ventas, productos y reportes a Excel o PDF con un clic.

Dashboard en vivo

Panel con refresh automático cada 5s. Gráficos de ventas 7 días.
